# Basic Calculator

A simple, responsive calculator built with HTML, CSS (Bootstrap 5), and JavaScript.

https://peguit-basic-calculator.netlify.app/
<img width="1592" height="800" alt="image" src="https://github.com/user-attachments/assets/4117ac3e-cdb0-4c35-98da-9b90b30289da" />


## Features

- Basic arithmetic operations: addition (+), subtraction (-), multiplication (×), division (÷)
- Clear (C) button to reset the calculator
- Backspace (←) button to delete the last character
- Error handling for invalid expressions
- Responsive design that works on different screen sizes

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Custom styling
- **Bootstrap 5.3.8** - Responsive grid system and components
- **JavaScript** - Calculator logic and interactivity
- **Google Fonts** - Figtree, Orbitron, and Roboto fonts

## How to Use

1. Open `index.html` in a web browser
2. Click on numbers and operators to build your expression
3. Click `=` to calculate the result
4. Click `C` to clear the display
5. Click `←` to delete the last character

```

## Installation

No installation required! Simply download the files and open `index.html` in your browser.
